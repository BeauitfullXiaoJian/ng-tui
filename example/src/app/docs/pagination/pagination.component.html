组件参数 @Component

选择器： ts-pagination

属性名称 类型 说明
pagination Pagination 分页参数，提供了当前页码，数据量，每页数据量
items Item[] 每页显示数据量选项列表，只有设置了这个属性，才会出现选项控件
goTitle string 跳转按钮的标题，只有设置了这个属性，才会出现跳转按钮
prevTitle string 上一页按钮的标题，只有设置了这个属性，才会出现此按钮
nextTitle string 下一页按钮的标题，只有设置了这个属性，才会出现此按钮
startTitle string 首页按钮的标题，只有设置了这个属性，才会出现此按钮
endTitle string 尾部按钮的标题，只有设置了这个属性，才会出现此按钮
color string 颜色设置，bootstrap的基础颜色[danger,light,info...]
outline 不需要设值 添加这个属性将会使用outline样式
sm 不需要设值 添加这个属性组件会变小
lg 不需要设值 添加这个属性组件会变大
事件名称 事件类型 说明
pageChange Pagination 当切换了页面会执行pageChange绑定的方法
