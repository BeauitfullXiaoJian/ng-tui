Pagination

Component
分页组件通常用于列表（表格）数据展示。
ts-pagination

Input Type Description
pagination Pagination 分页对象，保存维护当前页码，总数据，和其它分页相关参数
items Item[] 每页显示数据量选项列表，只有设置了这个属性，才会出现这个控件
goTitle string 跳转按钮的标题，只有设置了这个属性，才会出现跳转按钮
color string Color name, such as 'success', 'primary', etc.
Output Event Type Description
pageChange Pagination 当切换了页面会执行pageChange绑定的方法
Color
Example
数据总量： {{examplePage.total}} 当前页码： {{examplePage.page}} 每页数据量： {{examplePage.limit}} 总页数： {{examplePage.maxPage}} 分页参数： {{examplePage.pageData | json}}
