Button

Directive
Bootstrap基本按钮与带加载效果的按钮。
button[tsBtn]

Input Type Description
color string Color name, such as 'success', 'primary', etc.
disabled boolean Whether to disable the button
outline boolean Whether to use outline style
sm boolean Small size
lg boolean Large size
loading string Loading icon name
Output Event Type Description
submit Loader When the button sets the loading property, each time the button is pressed, the submit event is triggered, and $event is used to close the animation.
