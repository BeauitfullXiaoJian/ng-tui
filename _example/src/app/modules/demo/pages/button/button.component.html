按钮

1. 按钮指令可以帮助我们快速的创建一个按钮，支持颜色，尺寸等。 参考文档

2. 按钮指令会给button标签加上type="button"，这个会影响表单的提交。

3. 带动画的按钮通常用于表单提交，动画会在按钮按下的时候显示，需要在网络请求结束后关闭（无论请求成功或失败）。

指令参数 @Directive

选择器： button[tsBtn]

属性名称 类型 说明
color string 颜色设置，bootstrap的基础颜色[danger,light,info...]
disabled boolean 是否禁用按钮
outline boolean 不设值时为启用 添加这个属性按钮将会使用outline样式
sm boolean 不设值使用为启用 添加这个属性按钮会变小
lg boolean 不设值时为启用 添加这个属性按钮会变大
loading string 不设值使用默认图标（默认图标可以全局配置） 添加这个属性点击按钮会有加载动画，默认加载图标为loading
事件名称 事件类型 说明
submit Loader 当按钮设置了loading属性时，每次按下按钮会触发submit绑定的方法,$event用来关闭动画
按钮颜色
空心按钮
尺寸
